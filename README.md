# DqnutClans

## Commands
> <> is required  
> [ ] is optional

- /clans create **\<name>**
- /clans disband
- /clans list
- /clans info **\[clan]**
- /clans kick **\<player>**
- /clans ban **\<player>**
- /clans unban **\<player>**
- /clans mute **\<player>**
- /clans unmute **\<player>**
- /clans invite **\<player>**
- /clans join **\<player>**
- /clans leave
- /clans accept
- /clans deny
- /clans chat
- /clans setspawn
- /clans spawn
- /clans applications **\<accept|reject>** **\<player>**
